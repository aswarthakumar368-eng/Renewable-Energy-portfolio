# Renewable Energy Portfolio

A professional website portfolio showcasing three renewable energy system designs with block diagrams and detailed explanations.

## Activities Included

1. **Activity 1: Tidal Power Plant** - Coastal Village Electricity Generation System
2. **Activity 2: Biomass Biogas Generation System** - Dairy Farm with 100 Cattle
3. **Activity 3: Ocean Thermal Energy Conversion (OTEC)** - Electricity Production and Freshwater Generation

## Features

- Clean, modern design with responsive layout
- Left sidebar navigation for easy access to activities
- Detailed block diagrams using SVG graphics
- Comprehensive explanations (150-200 words) for each system covering:
  - Source of energy
  - Conversion process
  - Output/utilization
  - Real-world application relevance

## How to Deploy

### Option 1: GitHub Pages (Recommended - Free)

1. Create a GitHub repository
2. Upload all files to the repository
3. Go to Settings > Pages
4. Select source branch (usually `main` or `master`)
5. Your site will be live at: `https://yourusername.github.io/portfolio/`

### Option 2: Netlify (Free)

1. Go to [netlify.com](https://netlify.com)
2. Sign up/login
3. Drag and drop the `portfolio` folder
4. Your site will be live instantly with a URL like: `https://random-name.netlify.app`

### Option 3: Vercel (Free)

1. Go to [vercel.com](https://vercel.com)
2. Sign up/login
3. Import your repository or upload files
4. Deploy instantly

### Option 4: Google Sites / Wix / WordPress

If you prefer using a website builder:
1. Create a new site on your chosen platform
2. Copy the content from `index.html` into your pages
3. Upload images/diagrams separately
4. Set up navigation menu as left sidebar

### Option 5: Local Testing

Simply open `index.html` in your web browser to view the portfolio locally.
